# Stuxnet

**A Stuxnet clone made in Python for a research project**

## **Initialization**

```bash
pip3 install --upgrade pip && pip3 install -r requirements.txt
```

## **Build**

```bash
./build
```

## **Run** (must build beforehand)

```bash
./run
```

## **Build and run**

```bash
./build && ./run
```
